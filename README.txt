■はじめに

Originはゲームではなく玩具です。
楽しみ方はご自由に。

セーブといった機能はなく、終了後の状態維持はできません。
あしからず。

終了する場合はウインドウ右上の×ボタン以外に
Escキーで終了させることもできます。

正直まだいろいろ調整したいところだけども、
一旦公開、とさせていただきます。
どうなることやら。
